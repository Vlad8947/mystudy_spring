Существует два способа конфигурирования в Spring:
- через xml-файл конфигураций, расположенный в ресурсах
- через класс-конфигураций (AppConfig)

Примеры возможностей:
- указание директории для поиска аннотированных классов
- ручное прописывание бинов для контекста

Аннотации:
- @Configuration - класс-конфигураций (AppConfig)
- @Bean - создание бина в классе-конфигураций (AppConfig)
- @Component - указание создать объект-бин данного класса при сканировании директории (Box, ThingImpl)
- @Autowired - указание положить в эту переменную бин с таким же типом переменной (Box)
    required=false - при отсутствии бина данного типа, ничего не внедрять
    @Qualifier("beanName") - имя бина, который должен быть внедрён
